## Documentation

[www.guanguans.cn/coole](https://www.guanguans.cn/coole/)

## Requirement

* PHP >= 7.2

## Installation

``` bash
$ composer create-project coolephp/skeleton -vvv
```

## Quick start service

``` php
$ php coole serve
```

## Testing

``` bash
$ composer test
```
